## 指示

AI指示ファイル連携機能について、よくある質問への回答や使用方法、技術仕様を提供してください

## 文脈情報

このドキュメントは、Claude CodeおよびCursorにおいてAI指示ファイル（ai.md）を活用するための包括的なガイドです。この機能により、ユーザーは長文の指示や頻繁に変更される指示を効率的に管理し、プライバシーを保護しながらAIとの対話を向上させることができます。

## ルール

- プロジェクトのAI指示ファイル連携機能について、包括的で正確な情報を提供すること
- ユーザビリティとプライバシー保護の両面を考慮した説明を行うこと
- Claude CodeとCursor両方での使用方法を明記すること
- 技術的な制限や注意点も含めて説明すること

## 手順

1. AI指示ファイル連携機能の概要と目的を説明する
2. 基本的な使用方法を Claude Code と Cursor 別に説明する
3. よくある質問とその回答を提供する
4. 技術仕様と制限事項を明記する
5. プライバシー保護機能について説明する

## 出力形式

- マークダウン形式で出力してください
- 以下の構成で情報を整理してください：
  - ## 概要
  - ## 基本的な使用方法
  - ## よくある質問（FAQ）
  - ## 技術仕様
  - ## プライバシー保護
  - ## 注意点

## 入力データ

AI指示ファイル連携機能に関する質問や技術的な詳細

## 入出力例 (Few-shots)

**例1:**
入力:
```
AI指示ファイル機能の基本的な使い方を教えてください
```

出力:
```
## 概要
AI指示ファイル連携機能は、プロジェクトルートの `ai.md` ファイルに記述した指示内容を、Claude Code や Cursor で簡単に読み込める機能です。
## 基本的な使用方法
### Claude Code での使用
1. プロジェクトルートに `ai.md` ファイルを作成
2. ファイルに AI への指示を記述
3. Claude Code で `/ai` コマンドを実行
### Cursor での使用
1. プロジェクトルートに `ai.md` ファイルを作成
2. ファイルに AI への指示を記述
3. Cursor で `@ai` コマンドを実行
## プライバシー保護
`ai.md` ファイルは `.gitignore` に追加されているため、個人的なメモや機密情報も安全に記述できます。
```

**例2:**
入力:
```
ai.mdファイルの内容を変更した場合、すぐに反映されますか？
```

出力:
```
## よくある質問（FAQ）
### Q: ai.mdファイルの内容を変更した場合、すぐに反映されますか？
**A: はい、即座に反映されます。**
- セッション再起動は不要です
- 次回の `/ai` または `@ai` コマンド実行時に新しい内容が読み込まれます
- ファイル保存後、すぐにコマンドを実行できます
### Q: どのような内容をai.mdに書けますか？
**A: 以下のような内容を記述できます：**
- 長文の要件定義や仕様書
- プロジェクト固有のコーディング規約
- 個人的なTODOやメモ
- AI に対する詳細な指示や制約
- 機密性の高い情報（.gitignoreで保護されています）
```
