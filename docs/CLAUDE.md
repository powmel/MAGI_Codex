# docs/CLAUDE.md

## 文脈情報

docs 配下は、あらゆるドキュメントを管理するディレクトリです。
主に、Cursor など AI エージェントが読み取るシングルソースです。

## ディレクトリ構成

```
/docs
  /coding-rules          # コーディングルール（実装パターン、命名規則など）
  /development-workflows # 開発ワークフロー（Git操作、レビュー、リリースなど）
  /domain-knowledges     # ドメイン知識（仕様書、用語集、業務ルールなど）
```
